Validation & Flow Demo

Demonstrating strict column checking, flow control, and global row validation

Active Rules:
  • Allowed columns: makeId, typeLabel, volume, chipNumber (others will fail check)
  • Required columns: makeId (missing will fail check)
  • Flow: Force Check is ON (must check before import)
  • Flow: Prevent Start on Error is ON (cannot start if check fails)
  • Validation: chipNumber must be exactly 24 chars
  • Validation: If makeId is missing, typeLabel is required
